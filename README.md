# docker-nginx-phpfpm

# Usage

```
$ git clone git clone git@github.com:mochizukikotaro/docker-nginx-phpfpm.git
$ cd docker-nginx-phpfpm.git
$ docker-compose up
```

Access `localhost:8080`, then you can get phpinfo.
